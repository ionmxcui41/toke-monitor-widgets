### How to use
- These scripts are designed for Scriptable Widgets to count token usage.
- To use these scripts, you need to have a Scriptable app installed on your iOS device.

### Backend API
- The backend is powered by n8n, so you’ll need to set up your own n8n environment or create your own API to serve the data.
